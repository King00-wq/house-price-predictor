# house-price-predictor

A machine learning web application that predicts residential house prices using the Ames Housing dataset. Built with Flask, scikit-learn, and a dual-model architecture comparing Random Forest and Linear Regression.

---

## Project Overview

This project was developed as part of an AI/ML case study. It trains two regression models on the Kaggle House Prices dataset and serves predictions through a Flask web interface. A separate teacher-facing dashboard provides a full breakdown of model performance, feature importance, and the ML pipeline.

---

## Repository Structure

```
house-price-predictor/
├── app.py               Flask backend, prediction endpoint, routing
├── train_model.py       Training script for both models
├── train.csv            Ames Housing dataset (1460 rows, 81 columns)
├── model.pkl            Trained Random Forest model (auto-generated)
├── lr_model.pkl         Trained Linear Regression model + scaler (auto-generated)
├── metrics.json         Model evaluation results (auto-generated)
├── requirements.txt     Python dependencies
├── house_prices_analysis.ipynb  Exploratory data analysis notebook
└── templates/
    ├── index.html       Main prediction UI
    └── teacher.html     Model comparison dashboard
```

## Models

| Metric     | Random Forest     | Linear Regression |
|------------|-------------------|-------------------|
| R² Score   | 88.02%            | 79.72%            |
| MAE        | $18,833           | $24,530           |
| RMSE       | $30,312           | $39,443           |
| CV Mean    | 82.86%            | 76.06%            |

The Random Forest model uses 200 estimators with a max depth of 15. Linear Regression uses StandardScaler normalization before fitting.

---

## Features Used

The models are trained on 10 engineered features:

- OverallQual, GrLivArea, TotalSF, GarageCars, TotalBsmtSF
- YearBuilt, LotArea, TotalBath
- HouseAge (derived: 2024 - YearBuilt)
- RemodAge (derived: 2024 - YearRemodAdd)

## Setup and Usage

**1. Clone the repository**
```bash
git clone https://github.com/<your-username>/house-price-predictor.git
cd house-price-predictor
```

**2. Install dependencies**
```bash
pip install -r requirements.txt
```

**3. Train the models**
```bash
python train_model.py
```
This generates `model.pkl`, `lr_model.pkl`, and `metrics.json`.

**4. Run the application**
```bash
python app.py
```

**5. Open in browser**
- Prediction UI: http://localhost:5000
- Teacher dashboard: http://localhost:5000/teacher
